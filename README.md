# LangTest

LangChain agent demos with DeepSeek API integration.

## Files

- `agent_test.py`: Simple weather agent demo
- `.src/agent_demo.py`: Calculator agent demo with robust API key handling

## Setup

Set the `DEEPSEEK_API_KEY` environment variable before running the scripts.

```powershell
$env:DEEPSEEK_API_KEY="your-api-key-here"
```

## Usage

```bash
python agent_test.py
python .src/agent_demo.py
```
